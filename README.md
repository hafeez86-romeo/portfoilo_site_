# 👨‍💻 Personal Portfolio Website

A modern and responsive **Personal Portfolio Website** built using front-end technologies. This project is designed to showcase personal information, technical skills, projects, and contact details with an attractive and interactive user interface.

## ✨ About the Project

The Portfolio Website is developed to create a professional online presence. It displays the developer's profile, skills, projects, and achievements in an organized manner. The project focuses on implementing **HTML, CSS, and JavaScript** concepts to build a responsive and user-friendly website.

## 🔥 Features

✅ Responsive design for desktop, tablet, and mobile devices
✅ Modern dark and light theme switching option
✅ Smooth scrolling navigation
✅ Attractive hero section with profile display
✅ About Me section
✅ Technical skills showcase
✅ Project gallery with live demo and GitHub links
✅ Contact section for communication
✅ Interactive animations using JavaScript
✅ Mobile-friendly navigation menu
✅ Works on all modern web browsers

## 💻 Technologies Used

* **HTML5** – Used to create the structure and content of the webpage
* **CSS3** – Used for styling, layouts, animations, and responsive design
* **JavaScript** – Used to add interactivity, theme switching, and dynamic features
